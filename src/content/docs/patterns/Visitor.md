---
title: Visitor
description: Добавляет новые операции в иерархию классов объектов без изменения их исходного кода.
use_key: Сложные замороженные структуры объектов, требующие регулярного написания новых внешних алгоритмов обработки.
anti_use_key: Частое изменение самой структуры данных (добавление новых типов элементов потребует правки всех посетителей).
